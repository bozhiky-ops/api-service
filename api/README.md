## Introduction
The api-service project provides a simple RESTful API for managing data. It is designed to be highly scalable and performant.

## Getting Started
To get started with the api-service project, follow these steps:
* Clone the repository: `git clone https://github.com/api-service/api-service.git`
* Install dependencies: `npm install`
* Start the server: `npm start`

## API Endpoints
The api-service project provides the following API endpoints:
### GET /users
Returns a list of all users
### GET /users/:id
Returns a single user by id
### POST /users
Creates a new user
### PUT /users/:id
Updates a single user by id
### DELETE /users/:id
Deletes a single user by id
